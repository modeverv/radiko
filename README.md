$ ./radiko.sh -p BAYFM78
で聞くことができた。このBAYFM78みたいなキー（文字列）は、radiko.jpの各ラジオ局のページのURLで確認できる。w3mとか使えばサーバーでもok。

自分で各ラジオ局のキーをまとめてみたら以下のようになった。

NHK第1	JOAK
NHK第2	JOAB

FM横浜	YFM
文化放送	QRR
ラジオNIKKEI第1	RN1
ラジオNIKKEI第2	RN2
Jwave	FMJ
bayfm78	BAYFM78
TBSラジオ	TBS
NACK5	NACK5
TOKYOFM	FMT
ニッポン放送	LFR
